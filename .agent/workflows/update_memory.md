# Workflow: Update Project Memory

Этот воркфлоу предназначен для финальной фиксации знаний и изменений в документации проекта.

## Шаги выполнения

### 1. Загрузка навыка

// turbo

1. Загрузи навык `project-memory`:

   ```javascript
   mcp_skillport_load_skill({ skill_id: "project-memory" })
   ```

### 2. Анализ и Документирование

1. Проанализируй текущую сессию: выдели новые правила, паттерны и предпочтения пользователя.
2. Обнови **[.agent/GEMINI.md](../GEMINI.md)** если появились новые правила.
3. Создай подробный лог истории в **[docs/history/](../../docs/history/)**.
4. Внеси краткие записи в **[docs/CHANGELOG.md](../../docs/CHANGELOG.md)**.
5. Обнови **[docs/BACKLOG.md](../../docs/BACKLOG.md)** на основе обсужденных планов.

### 3. Верификация

// turbo

1. Запусти скрипт проверки документации:

   ```bash
   node scripts/verify_docs.js
   ```

### 4. Отчет и Подтверждение

1. Предоставь пользователю ссылки на обновленные документы.
2. **СТОП**: Дождись явного одобрения от пользователя, прежде чем считать задачу по обновлению памяти завершенной.
